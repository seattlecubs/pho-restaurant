Welcome to HTML Basics!

1. What is HTML?

HTML stands for HyperText Markup Language. It's the standard markup language for creating web pages. HTML uses tags to structure content.

2. Basic HTML Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Web Page</title> </head> <body> <h1>Hello World!</h1> <p>This is my first paragraph.</p> </body> </html>

6. HTML Attributes

HTML elements can have attributes that provide additional information:

<a href="https://www.example.com" target="_blank">External Link</a> <img src="photo.jpg" alt="Photo description" width="400" height="300"> <div class="container" id="main-content">Content here</div>

7. Semantic HTML

Use semantic elements to give meaning to your content:

<header>Page header</header> <nav>Navigation menu</nav> <main> <article>Main article content</article> <section>Section content</section> </main> <aside>Sidebar content</aside> <footer>Page footer</footer>

8. Key Points to Remember

  • HTML tags usually come in pairs: opening <tag> and closing </tag>
  • Some tags are self-closing: <img>, <br>, <hr>
  • Always close your tags in the reverse order you opened them
  • Use lowercase for tag names and attribute names
  • Always include alt text for images for accessibility
  • Use semantic HTML for better structure and SEO
